## Description
This project demonstrates basic software engineering standards, including environment setup, directory organization, and version control using GitHub. The Python script included is a **Student Record System** that utilizes various data structures such as Dictionaries, Lists, and Tuples to manage and display student information.

## Project Structure
The repository is organized following standard software development practices:
- `src/`: Contains the main Python source code (`main.py`).
- `tests/`: Contains screenshots of the program execution and test runs.
- `README.md`: Project documentation and instructions.

## Tools Used
- **OneCompiler:** Online Integrated Development Environment (IDE) used for writing and testing the Python script.
- **GitHub:** Used for version control and maintaining a history of meaningful commits.
- **Python 3:** The programming language used for the logic implementation.

## How to Run the Activity
1. Open the `src/main.py` file.
2. Copy the code into any Python 3 environment (like OneCompiler or a local Python interpreter).
3. Run the script to see the formatted student record output.
4. Review the `tests/` folder to see the expected output evidence.

## Test Cases
- **Test 1:** Default execution displaying initial student data.
- **Test 2:** Modified data test (updated name and year level).
- **Test 3:** Collection update test (added new skills to the list).
